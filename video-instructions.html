üé¨ Hotel Video Background Setup

This guide will help you set up a beautiful video background for your hotel website homepage.

üìπ Video Requirements

Place Your Video Files Here

Add your hotel promotional video to this directory:

assets/videos/

Required files:

  • hotel-bg.mp4 - Main video file (MP4 format)
  • hotel-bg.webm - WebM format for better browser compatibility
  • hero-poster.jpg - Poster image shown before video loads

üé• Video Specifications

üí° Recommended Video Specifications:
  • Resolution: 1920x1080 (Full HD) or 1280x720 (HD)
  • Aspect Ratio: 16:9
  • Duration: 15-30 seconds (looping)
  • File Size: Under 20MB for optimal loading
  • Format: MP4 (H.264 codec) and WebM (VP8/VP9)

üé¨ HTML Video Tag

The video is implemented in public/index.php using this code:

<!-- Video Background -->
<video autoplay muted loop playsinline class="video-background" 
       poster="assets/images/hero-poster.jpg">
    <source src="assets/videos/hotel-bg.mp4" type="video/mp4">
    <source src="assets/videos/hotel-bg.webm" type="video/webm">
    <!-- Fallback image if video doesn't load -->
    <img src="assets/images/hero-fallback.jpg" 
         alt="Luxury Hotel Background" class="video-fallback">
</video>

<!-- Video Overlay for better text readability -->
<div class="video-overlay"></div>

üìÅ Video Attributes Explained

  • autoplay - Video starts automatically
  • muted - Video plays without sound (required for autoplay in most browsers)
  • loop - Video repeats continuously
  • playsinline - Video plays inline on mobile devices (iOS)
  • poster - Image shown while video loads

üé® Video Overlay CSS

The overlay darkens the video to make text more readable:

.video-overlay {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: linear-gradient(to bottom, 
                                rgba(0, 0, 0, 0.4), 
                                rgba(0, 0, 0, 0.7));
    z-index: 1;
}

üõ†Ô∏è Converting Your Video

If you have a video in a different format, you can convert it using:

  • Online: CloudConvert, Convertio
  • Desktop: HandBrake, VLC Media Player, FFmpeg
  • Command Line (FFmpeg): 
    ffmpeg -i input.mp4 -c:v libx264 -crf 23 -c:a aac -movflags +faststart output.mp4
ffmpeg -i input.mp4 -c:v libvpx -b:v 1M -c:a libvorbis output.webm

üñºÔ∏è Poster & Fallback Images

Create these images for the best experience:

  • hero-poster.jpg - First frame of the video (1920x1080)
  • hero-fallback.jpg - Static background image for older browsers

‚ö° Performance Tips

Optimization Tips:
  • Use compressed video files to reduce loading time
  • Host videos on a CDN for better performance
  • Implement lazy loading for better page speed
  • Consider using a video hosting service (YouTube, Vimeo) for large files
  • Test on different devices and connection speeds

üéØ Alternative: Static Background

If you prefer a static background instead of video, replace the video tag with:

<div class="hero-background">
    <img src="assets/images/hero-bg.jpg" alt="Hotel Background">
</div>

And update the CSS:

.hero-background {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    overflow: hidden;
}

.hero-background img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}
